Social Network API

This is a social media API that uses mongodb (a NoSQL database) and mongoose that will allow your website to handle large amounts of unstructured data. When you open API GET routes for users and thoughts the data for each of these routes is displayed in a formatted JSON. If you test the API POST, PUT, and DELETE routes you will be able to successfully create, update, and delete users and thoughts in the database. By testing the API POST and DELETE routes you will also be able to create and delete reactions to thoughts, and add or remove friends to a user's friend list.
